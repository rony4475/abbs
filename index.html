Remote Desktop From Github - Fleter

This is how you create your own remote desktop from github

  • Fork My Repository
  • In "host.ps1" please change ngrok auth token to your auth token
  • Go to action
  • Click at "RDP Creator" in action list
  • Click "run workflow" grey background button
  • click button "run workflow" green background button
  • Open your ngrok tunnels agents by click here
  • If you see theres have some IP address available click it and join with that ip address (the ip is started by "tcp://")
  • Else if its dosnt exist, just refresh multiple time until its showing and make sure the workflow still running
